# Installation

Enable venv

```
python3.7 -m venv django-project
source django-project/bin/activate
pip install Django
```

Check version

```
python -m django --version
3.0.6
```

Start a project

```
django-admin startproject jarlok
```

Create application
```
python manage.py startapp polls
```
