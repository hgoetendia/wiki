# Time sleep


```python
#!/usr/bin/python
import time

print "Starting : %s" % time.ctime()
time.sleep( 15 )
print "Ending : %s" % time.ctime()
#result:
#Start : Tue Feb 17 11:19:18 2019
#End : Tue Feb 17 11:19:33 2019
```
